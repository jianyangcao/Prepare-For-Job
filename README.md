# Prepare-For-Job
# SQL + Python Foundation (DS / MLE)

A project-driven learning repo (2026/01/14 - 2026/09/30).
Focus: SQL templates + reproducible Python analysis + weekly insight cards.

## Repo Structure
- /sql_templates: reusable SQL template pack
- /python_notebooks: notebooks & scripts
- /insight_cards: weekly 1-page findings (evidence + explanation + next steps)
- /projectA: Funnel + Retention + Segmentation diagnosis (deliverable before March)
- /projectB: Modeling or Data pipeline (deliverable by June)
- /interview_prep: SQL drills, AB test cards, mock notes

## How to Run
1) Create env
- python -m venv .venv && source .venv/bin/activate
- pip install -r requirements.txt

2) Notebooks
- jupyter lab
